===============================================================================

===============================================================================

    virtualenv arkestra
    cd arkestra/
    source bin/activate

                    Установить Arkestra c зависимостями
===============================================================================

::
    pip install -e git+https://github.com/evildmp/Arkestra.git@develop#egg=Arkestra
